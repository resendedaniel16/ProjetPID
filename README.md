# Projet Reservations – Spring Boot

Projet Spring Boot réalisé dans le cadre du cours de Java / Spring.

## Technologies
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- MySQL
- Flyway

## Lancement du projet
1. Démarrer MySQL
2. Créer la base de données `reservations`
3. Lancer l'application depuis IntelliJ ou avec :
   mvn spring-boot:run

## Fonctionnalités
- Gestion des artistes
- Gestion des spectacles
- Réservations
- API REST (exemple sur Artist)
